# Run and deploy your app

This contains everything you need to run your app locally.

View your app : https://bhogendra-jewellers.vercel.app/

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your google
3. Run the app:
   `npm run dev`
